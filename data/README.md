# Flight Data Decode 2   

Flight Data Decode 2, 解析,解码,译码 原始QAR数据 raw.dat 文件。ARINC 429 573 717. FOQA, arinc429, arinc717. Airfase.  

当前目录是 数据目录。  
* `1024.PRM` 截取的配置文件。从 Airfase中导出的。  
* `output*`  qar_decode 的输出。  
* `raw.dat`  截取的原始数据，aligned格式。
